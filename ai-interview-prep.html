想定質問の生成

基本的な質問生成プロンプト

企業・職種特化質問生成

以下の条件で面接想定質問を作成してください：

【企業情報】
- 企業名：〇〇株式会社
- 業界：[具体的な業界]
- 事業内容：[主要事業]
- 企業文化：[社風・価値観]
- 最近のニュース：[直近の動向]

【職種情報】
- 職種：[希望職種]
- 部署：[配属予定部署]
- 求められるスキル：[必要な能力]
- キャリアパス：[将来の道筋]

【応募者情報】
- 学歴：[大学・学部・専攻]
- 経験：[インターン・アルバイト等]
- 資格・スキル：[保有資格]
- 志望動機：[簡潔な理由]

【出力要件】
1. 基本質問（5問）
2. 企業特化質問（5問）
3. 職種特化質問（5問）
4. 圧迫・深掘り質問（3問）
5. 逆質問への準備（3問）

各質問に難易度（★〜★★★）と回答のポイントを併記

質問カテゴリ別生成

自己紹介・自己PR

  • 「簡単に自己紹介をお願いします」
  • 「あなたの強みを教えてください」
  • 「なぜその専攻を選んだのですか？」
  • 「学生時代に最も力を入れたことは？」

志望動機・企業理解

  • 「なぜ当社を志望するのですか？」
  • 「他社との違いをどう考えますか？」
  • 「当社の事業について知っていることは？」
  • 「10年後、当社はどうなっていると思いますか？」

職種・業務理解

  • 「この職種を選んだ理由は？」
  • 「どのような業務をイメージしていますか？」
  • 「必要なスキルは何だと思いますか？」
  • 「困難な状況にどう対処しますか？」

将来・キャリア

  • 「5年後はどうなっていたいですか？」
  • 「キャリアプランを教えてください」
  • 「転勤は可能ですか？」
  • 「将来起業したいと思いますか？」

回答準備とブラッシュアップ

回答作成支援プロンプト

STAR法による回答構築

以下の面接質問に対してSTAR法で回答を作成してください：

【質問】
「学生時代に最も困難だった経験とそれをどう乗り越えたか教えてください」

【あなたの経験】
- 状況：[具体的な状況説明]
- 課題：[直面した問題・困難]
- 行動：[とった具体的な行動]
- 結果：[達成した成果・学び]

【回答要件】
1. STAR法の構成を明確に
2. 具体的な数字や事実を含める
3. 感情や心境の変化も表現
4. 学びや成長を強調
5. 2-3分で話せる長さ（300-400文字）
6. 企業の求める人材像との関連性

【出力形式】
- 完成した回答文
- 各部分（S-T-A-R）の文字数配分
- 話す際のポイント・注意点
- より良くするための改善提案

回答の構造化技法

STAR法

  • Situation：状況の説明（30秒）
  • Task：課題・目標の明確化（30秒）
  • Action：具体的な行動（60-90秒）
  • Result：結果・学び（30-60秒）

PREP法

  • Point：結論・要点（20秒）
  • Reason：理由・根拠（60秒）
  • Example：具体例・体験（60秒）
  • Point：結論の再確認（20秒）

DESC法

  • Describe：状況の描写（40秒）
  • Express：感情・思いの表現（40秒）
  • Specify：具体的な行動（60秒）
  • Consequences：結果・影響（40秒）

AI模擬面接の活用

模擬面接セッションの設計

AI面接官設定プロンプト

あなたは以下の企業の面接官として模擬面接を行ってください：

【企業設定】
- 企業名：〇〇株式会社
- 業界：[業界名]
- 面接官の役職：[人事部長/現場責任者等]
- 面接形式：[一次面接/最終面接]
- 面接時間：30分

【面接官の特徴】
- 年齢：40代
- 性格：[厳格/親しみやすい/論理的等]
- 重視する点：[協調性/論理性/積極性等]
- 質問スタイル：[深掘り型/幅広く質問型]

【面接の進行】
1. 自己紹介から開始
2. 志望動機の確認
3. 学生時代の経験について深掘り
4. 企業・職種理解の確認
5. 逆質問の受付
6. 面接終了とフィードバック

【評価観点】
- 回答の論理性と一貫性
- 企業理解の深さ
- コミュニケーション能力
- 熱意と誠実さ
- 将来性と成長意欲

一問ずつ質問し、私の回答に対して適切なフォローアップ質問やフィードバックを提供してください。

面接シナリオ別対策

一次面接

特徴
  • 基本的な情報確認
  • 人柄やコミュニケーション力
  • 志望度の測定
  • 基本的なマナー確認
対策ポイント
  • 明るく簡潔な自己紹介
  • 基本的な志望動機の準備
  • 学生時代のエピソード
  • 適切な敬語と姿勢

グループ面接

特徴
  • 他の応募者との比較
  • 協調性と積極性のバランス
  • 限られた時間での印象づけ
  • ディスカッション能力
対策ポイント
  • 簡潔で印象的な発言
  • 他者の意見への反応
  • 適度な積極性
  • チームワークの演出

最終面接

特徴
  • 経営陣との面接
  • 企業文化との適合性
  • 長期的なビジョン確認
  • 入社意欲の最終確認
対策ポイント
  • 企業理念への深い理解
  • 具体的な将来ビジョン
  • 他社との違いの明確化
  • 強い入社意欲の表明

Web面接

特徴
  • 技術的な準備が必要
  • 画面越しのコミュニケーション
  • 環境要因の影響
  • 非言語的要素の制限
対策ポイント
  • 機材と環境の事前確認
  • カメラ目線での話し方
  • 表情とジェスチャーの意識
  • 音声の明瞭さ

    • フィードバック分析と改善

    AI評価項目

    コミュニケーション

    • 明瞭さ：発声・滑舌・話すスピード
    • 論理性：話の筋道・構成・結論
    • 簡潔性：要点の整理・時間管理
    • 表現力：豊かな語彙・具体性

    内容・知識

    • 企業理解：事業・理念・文化の理解度
    • 職種理解：業務内容・必要スキルの把握
    • 自己分析：強み・経験の整理・表現
    • 将来性：ビジョン・成長意欲

    態度・姿勢

    • 熱意：志望度・意欲の表現
    • 誠実さ：正直・素直な対応
    • 柔軟性：質問への適応・対応力
    • 礼儀：マナー・敬語・姿勢

    改善アクションプラン

    フィードバック分析プロンプト

    以下の模擬面接の回答を分析し、改善提案を行ってください：

【質問】
[面接質問]

【回答】
[あなたの回答]

【分析観点】
1. 内容の適切性（企業・職種との関連性）
2. 構成の論理性（STAR法等の活用）
3. 具体性（数字・事実の含有）
4. 表現の自然さ（AIっぽくない表現）
5. 時間の適切性（長すぎる・短すぎる）

【出力要件】
- 良い点（3つ）
- 改善点（3つ）
- 具体的な改善案（修正版回答例）
- 練習方法の提案
- 関連する追加質問への準備アドバイス

評価は5段階で採点し、理由も併記してください。

    業界別面接対策

    業界特有の質問傾向

    IT・技術系

    • 「なぜプログラミングに興味を持ったのですか？」
    • 「どのような技術を学んでいますか？」
    • 「AIやクラウドについてどう思いますか？」
    • 「技術の進歩についていく自信はありますか？」

    金融・コンサル

    • 「最近の経済ニュースで気になることは？」
    • 「論理的思考力をどう身につけましたか？」
    • 「ストレス耐性はありますか？」
    • 「なぜその業界を選んだのですか？」

    営業・販売

    • 「人とのコミュニケーションで心がけることは？」
    • 「断られても諦めない理由は？」
    • 「目標達成のために何をしますか？」
    • 「お客様第一をどう実践しますか？」

    メーカー・製造

    • 「ものづくりの魅力は何だと思いますか？」
    • 「品質管理の重要性をどう考えますか？」
    • 「チームでの作業経験はありますか？」
    • 「海外展開についてどう思いますか？」

    行動面接質問対策

    行動面接の特徴と対策

    行動面接とは

    過去の具体的な行動や経験を通じて、応募者の能力や性格を評価する面接手法。「どうしましたか？」「どう考えましたか？」といった質問が特徴。

    評価される要素

    • 問題解決能力
    • リーダーシップ
    • チームワーク
    • コミュニケーション能力
    • ストレス耐性
    • 学習能力

    典型的な行動面接質問

    問題解決

    • 「困難な問題に直面した時、どう対処しましたか？」
    • 「期限に間に合わない状況でどう行動しましたか？」
    • 「予想外の事態にどう対応しましたか？」

    チームワーク

    • 「チームで意見が対立した時、どうしましたか？」
    • 「チームメンバーが協力してくれない時は？」
    • 「チームの成果を上げるために何をしましたか？」

    リーダーシップ

    • 「リーダーとして最も困難だった経験は？」
    • 「メンバーのモチベーションをどう上げましたか？」
    • 「方針を変更する必要があった時の対応は？」

    成長・学習

    • 「失敗から何を学びましたか？」
    • 「新しいスキルをどう身につけましたか？」
    • 「フィードバックをどう活用しましたか？」

    面接ストレス管理

    AI活用によるメンタル準備

    心理的準備

    • ポジティブ思考の練習：AIとの成功体験の積み重ね
    • 不安の可視化：心配事の整理と対策準備
    • リラクゼーション：呼吸法や瞑想の練習
    • 自信の構築：強みの再確認と実績の整理

    技術的準備

    • 十分な練習：AIとの反復練習で慣れる
    • 想定外への準備：様々なパターンの練習
    • 時間感覚の習得：回答時間の体感練習
    • 非言語スキル：表情・姿勢・声のトーン

    当日の準備

    • 体調管理：十分な睡眠と栄養
    • 時間的余裕：早めの到着と準備時間
    • 持ち物確認：必要書類と身だしなみ
    • 最終確認：重要ポイントの復習

    面接成功のコツ

    自然体で

    過度に緊張せず、準備した内容を自然に話す

    相手の話を聞く

    質問の意図を理解し、適切に回答する

    熱意を伝える

    企業への興味と入社への意欲を表現する

    対話を心がける

    一方的ではなく、相互のコミュニケーションを意識
